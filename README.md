## 适合人群

- 对 AI 感兴趣的初学者
- 希望系统化学习 AI 的开发者
- 准备转型 AI 领域的技术人员

## 前置知识

- 基础编程能力（推荐 Python）
- 高中及以上数学基础（线性代数、概率统计、微积分有帮助但非必须）
