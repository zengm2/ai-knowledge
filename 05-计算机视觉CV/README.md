# 05 - 计算机视觉 (CV)

> 计算机视觉是人工智能中最直观、最令人兴奋的领域之一——它赋予机器"看"的能力。从手机上的人脸解锁，到自动驾驶汽车识别行人，再到 AI 绘画生成艺术作品，计算机视觉正在深刻改变我们的生活。

---

## 5.1 什么是计算机视觉

### 定义：让计算机"看懂"图像和视频

计算机视觉（Computer Vision，简称 CV）是一门研究如何让计算机从图像或视频中**提取有意义的信息**并做出决策的学科。

打一个比喻：人类看到一张猫的照片，大脑会瞬间理解"这是一只猫"。而计算机看到的只是一堆数字——计算机视觉的目标，就是教会计算机像人一样理解这些数字背后的含义。

### 图像在计算机中的表示

在计算机中，一张图像本质上是一个**数字矩阵**：

- **灰度图像**：每个像素用一个 0~255 的数值表示亮度（0 为黑色，255 为白色）。一张 256×256 的灰度图就是一个 256×256 的二维矩阵。
- **彩色图像 (RGB)**：每个像素由三个通道组成——**R（红）、G（绿）、B（蓝）**，每个通道的取值范围都是 0~255。因此一张 256×256 的彩色图像，其数据形状为 `256 × 256 × 3`。

可以这样理解：一张彩色照片就像三张"滤镜纸"叠在一起，分别记录了红色、绿色和蓝色分量的强度，三者混合后形成丰富的色彩。

### CV 的核心挑战

计算机视觉面临许多人类视觉系统能轻松应对、但机器难以处理的挑战：

- **视角变化**：同一个物体从不同角度看起来完全不同
- **光照变化**：同一场景在白天和夜晚差异巨大
- **遮挡问题**：目标被其他物体部分遮挡
- **类内差异**：同一类别的物体外观差异大（比如不同品种的狗）
- **背景干扰**：目标与背景难以区分

这些挑战推动着 CV 技术不断进步，从早期的手工特征方法发展到如今强大的深度学习方法。

## 5.2 图像分类

### 任务定义

图像分类是计算机视觉中最基础的任务：**给定一张图像，判断它属于哪个类别**。例如，输入一张动物照片，输出"猫"或"狗"。

### 从手工特征到深度特征

CV 的发展经历了两个大阶段：

1. **传统方法（手工特征时代）**：研究者需要手动设计特征提取算法（如 SIFT、HOG），然后用机器学习分类器（如 SVM）进行分类。这就像给计算机写了一本"识别手册"，告诉它该关注哪些纹理、边缘、颜色。
2. **深度学习方法（自动学习特征）**：卷积神经网络（CNN）能够自动从数据中学习特征，不再需要人工设计。这就像让计算机自己"阅读"成千上万张图片后，总结出识别规律。

### ImageNet 竞赛的里程碑意义

**ImageNet** 是一个包含超过 1400 万张图片、2 万多个类别的大型图像数据集。ImageNet 大规模视觉识别挑战赛（ILSVRC）是 CV 领域最重要的竞赛之一。

2012 年，**AlexNet** 以大幅领先的优势赢得比赛，将 Top-5 错误率从 26% 降至 16%，标志着深度学习在 CV 领域的崛起，开启了一个新时代。

### 经典网络演进

| 网络 | 年份 | 核心贡献 |
|------|------|----------|
| **AlexNet** | 2012 | 首次在大规模图像分类中证明深度 CNN 的威力，使用 ReLU 激活函数和 Dropout |
| **VGG** | 2014 | 证明了"更深的网络效果更好"，采用统一的 3×3 小卷积核堆叠策略 |
| **GoogLeNet (Inception)** | 2014 | 引入 Inception 模块，在同一层使用不同尺寸的卷积核并行提取特征，提高效率 |
| **ResNet** | 2015 | 提出**残差连接（Skip Connection）**，解决了深层网络难以训练的问题，网络深度达到 152 层甚至更深 |
| **EfficientNet** | 2019 | 提出**复合缩放策略**，同时优化网络深度、宽度和分辨率，以更少的参数达到更高的精度 |

> 💡 **关键理解**：ResNet 的残差连接思想尤为重要——它让信息可以"跳过"某些层直接传递，就像在高速公路上开辟了快速通道，让深层网络也能顺畅地学习。这一思想后来深刻影响了 Transformer 等架构的设计。

## 5.3 目标检测

### 任务定义：定位 + 分类

如果说图像分类回答的是"**图里有什么？**"，那么目标检测回答的是"**图里有什么？在哪里？**"。目标检测需要同时完成两件事：

1. **定位**：用矩形框（Bounding Box）标出目标位置
2. **分类**：判断框内的目标属于哪个类别

### 两阶段方法（Two-Stage）

两阶段方法先生成候选区域，再对候选区域进行分类，精度较高但速度偏慢：

- **R-CNN (2014)**：先用选择性搜索生成约 2000 个候选区域，再逐一用 CNN 提取特征并分类。开创性工作，但速度极慢。
- **Fast R-CNN (2015)**：改进为先对整张图提取特征图，再在特征图上裁剪候选区域，避免重复计算，速度大幅提升。
- **Faster R-CNN (2015)**：引入**区域建议网络（RPN）**，让候选区域的生成也由神经网络完成，实现了端到端训练，成为目标检测的经典框架。

### 单阶段方法（One-Stage）

单阶段方法直接在特征图上预测目标的类别和位置，速度更快，适合实时应用：

- **YOLO 系列**（You Only Look Once）：将图像划分为网格，每个网格直接预测目标。YOLO 系列从 v1 发展到 v8 及之后版本，不断在速度和精度之间取得更好的平衡，是工业界应用最广泛的检测模型之一。
- **SSD**（Single Shot MultiBox Detector）：在多个不同尺度的特征图上进行检测，能够同时检测大目标和小目标。

### Anchor-Free 方法

传统检测方法依赖预设的"锚框"（Anchor），而 Anchor-Free 方法直接预测目标的关键点或中心点，设计更简洁：

- **CornerNet**：检测目标的左上角和右下角
- **CenterNet**：检测目标的中心点，再回归宽高
- **FCOS**：基于逐像素预测，完全摆脱锚框

### 评估指标

- **IoU（交并比）**：预测框与真实框的重叠程度，计算公式为两者交集面积除以并集面积。IoU 越高，定位越准。
- **mAP（平均精度均值）**：综合衡量检测模型在所有类别上的精度和召回率的指标。mAP 越高，模型整体性能越好。

## 5.4 图像分割

图像分割是比目标检测更精细的任务——它要求模型判断**每一个像素**属于什么类别。

### 三种分割任务

| 类型 | 说明 | 比喻 |
|------|------|------|
| **语义分割** | 为每个像素分配类别标签，但不区分同类别的不同个体 | 给图片中所有"人"像素涂上同一种颜色 |
| **实例分割** | 不仅分类，还区分同类别的不同个体 | 给图片中每个"人"涂上不同的颜色 |
| **全景分割** | 语义分割 + 实例分割的统一，同时处理"可数物体"和"不可数背景" | 既区分每个人，也标注出天空、道路等背景区域 |

### 关键模型

**FCN（全卷积网络，2015）**

FCN 是深度学习语义分割的开山之作。它将分类网络中的全连接层替换为卷积层，使网络能够接受任意尺寸的输入并输出与输入同尺寸的分割图。核心思想是：分类网络看的是"整体"，FCN 看的是"每个位置"。

**U-Net（2015）**

U-Net 最初为医学图像分割设计，其标志性特征是**对称的编码器-解码器结构**和**跳跃连接**：

- **编码器**（下采样路径）：逐步提取高层语义特征，看到"大局"
- **解码器**（上采样路径）：逐步恢复空间分辨率，还原"细节"
- **跳跃连接**：将编码器的细节信息传递给解码器，弥补上采样丢失的空间信息

这种设计使 U-Net 即使在少量标注数据下也能取得出色效果，至今仍广泛应用于医学影像分析。

**Mask R-CNN（2017）**

Mask R-CNN 在 Faster R-CNN 的基础上增加了一个**分割分支**，在进行目标检测（框 + 类别）的同时，为每个检测到的实例生成像素级的分割掩码。它是实例分割领域的里程碑模型。

## 5.5 图像生成

图像生成是 CV 中最"酷"的方向之一——让计算机创造出逼真的、甚至从未存在过的图像。

### 生成对抗网络 (GAN)

GAN（2014 年由 Ian Goodfellow 提出）的核心思想极为巧妙，包含两个神经网络的"对抗博弈"：

- **生成器（Generator）**：试图生成以假乱真的图像，就像一个"造假画家"
- **判别器（Discriminator）**：试图分辨真假图像，就像一个"鉴定专家"

两者在训练中相互对抗、共同进步：造假画家越来越擅长伪造，鉴定专家越来越擅长识别，最终生成器能够产生极其逼真的图像。GAN 的经典变体包括 DCGAN、StyleGAN、CycleGAN 等。

### 变分自编码器 (VAE)

VAE 采用"编码-解码"的框架：

- **编码器**：将输入图像压缩到一个**潜在空间**（Latent Space）的概率分布
- **解码器**：从潜在空间中采样并重建图像

VAE 的优势在于其潜在空间是连续且有结构的，可以通过在潜在空间中"漫步"来实现图像的平滑过渡和属性编辑。

### 扩散模型 (Diffusion Model)

扩散模型是当前图像生成领域的主流技术，也是 **Stable Diffusion、DALL-E 2/3、Midjourney** 等热门 AI 绘画工具的核心引擎。

其基本原理可以用一个比喻来理解：

- **前向过程（加噪）**：想象一张清晰的照片逐步被撒上越来越多的噪点，最终变成纯噪声——就像一幅画被逐渐模糊直到看不清
- **反向过程（去噪）**：模型学习从纯噪声中一步步去除噪点，逐渐还原出清晰的图像——就像一位画家从一团迷雾中逐笔勾勒出一幅画

扩散模型通过学习"如何去噪"，掌握了图像的分布规律，从而能够生成高质量、多样化的图像。

### 图像生成的应用

- **AI 绘画**：根据文字描述生成艺术作品（Text-to-Image）
- **图像修复（Inpainting）**：自动填补图像中缺失或损坏的区域
- **超分辨率（Super Resolution）**：将低分辨率图像增强为高分辨率，让模糊的图片变清晰
- **风格迁移**：将一种艺术风格应用到另一张图像上，如将照片变成梵高风格的画作
- **图像编辑**：基于文字指令对图像进行局部或全局编辑

## 5.6 多模态视觉

多模态视觉是近年来最火热的研究方向之一，核心思想是**打通视觉与语言的壁垒**，让模型同时理解图像和文本。

### 图文匹配：CLIP

**CLIP**（Contrastive Language-Image Pre-training，OpenAI，2021）是多模态视觉的里程碑：

- **核心思想**：用对比学习的方式，让图像编码器和文本编码器学会将匹配的图文对映射到相近的向量空间
- **训练数据**：在互联网上收集的 4 亿个图文对上进行预训练
- **强大能力**：无需针对特定任务微调，就能进行零样本（Zero-Shot）图像分类——只需给出类别的文字描述，CLIP 就能判断图像属于哪个类别

CLIP 的出现极大推动了后续的多模态模型发展，也是 Stable Diffusion 等生成模型中文本理解能力的重要基础。

### 视觉问答 (VQA)

VQA（Visual Question Answering）任务要求模型根据一张图像回答自然语言问题。例如：

- 输入图像：一张厨房的照片 + 问题："桌上有几个苹果？"
- 模型输出："3 个"

这要求模型同时具备视觉理解和语言理解能力。

### 图像描述 (Image Captioning)

Image Captioning 要求模型为一张图像自动生成一段自然语言描述。例如：

- 输入：一张海边日落的照片
- 输出："一个人在金色的夕阳下沿着沙滩散步"

这是连接视觉与语言的经典任务，广泛应用于图片搜索、无障碍辅助等场景。

### 视觉大模型的趋势

随着大语言模型（LLM）的爆发，视觉大模型正在快速发展：

- **GPT-4V / GPT-4o**：OpenAI 的多模态大模型，能够理解和分析图像内容
- **Gemini**：Google 的原生多模态模型，从设计之初就融合了视觉和语言
- **LLaVA**：开源的视觉语言模型，将视觉编码器与大语言模型连接
- **Qwen-VL / InternVL**：国内优秀的开源多模态大模型

核心趋势是：**视觉不再是一个独立的模块，而是与语言、推理能力深度融合**，朝着通用人工智能（AGI）的方向迈进。

## 5.7 学习建议与延伸阅读

### 推荐学习路径

1. **打好基础**：先学习 Python 编程和 NumPy/PyTorch 基础操作
2. **理解 CNN**：从卷积、池化、激活函数等基本概念入手，手动实现一个简单的图像分类器
3. **经典论文**：阅读 AlexNet、ResNet、YOLO 等经典论文，理解核心设计思想
4. **动手实践**：在 MNIST、CIFAR-10 等小数据集上训练模型，逐步过渡到更复杂的任务
5. **跟进前沿**：关注扩散模型、多模态视觉等前沿方向

### 推荐资源

- **课程**：
  - Stanford CS231n：Convolutional Neural Networks for Visual Recognition（经典中的经典）
  - 李飞飞团队的计算机视觉课程
- **书籍**：
  - 《动手学深度学习》(Dive into Deep Learning) — 李沐等著
  - 《深度学习》(Deep Learning) — Ian Goodfellow 等著
- **实践平台**：
  - Kaggle 计算机视觉竞赛
  - Papers With Code（查看最新论文及其代码实现）
  - Hugging Face（丰富的预训练模型和数据集）
- **经典论文**：
  - AlexNet: *ImageNet Classification with Deep Convolutional Neural Networks* (2012)
  - ResNet: *Deep Residual Learning for Image Recognition* (2015)
  - YOLO: *You Only Look Once: Unified, Real-Time Object Detection* (2016)
  - U-Net: *U-Net: Convolutional Networks for Biomedical Image Segmentation* (2015)
  - GAN: *Generative Adversarial Networks* (2014)
  - CLIP: *Learning Transferable Visual Models From Natural Language Supervision* (2021)
  - Denoising Diffusion Probabilistic Models (DDPM, 2020)

### 写在最后

计算机视觉是一个既有深厚理论基础、又有广泛实际应用的领域。从基础的图像分类到前沿的多模态大模型，CV 技术正在以惊人的速度发展。对于初学者来说，最重要的是**动手实践**——理论固然重要，但只有通过亲自训练模型、调试代码，才能真正掌握这些技术。

> 🎯 **核心心法**：先跑通一个完整的项目（哪怕是最简单的手写数字识别），建立信心和直觉，然后再逐步深入理解每个组件的原理。学习 CV 的过程，就像训练神经网络一样——需要不断迭代、持续优化！
